# JPGSU-DL — Python Image Scraper & Downloader
a lightweight Python script that scrapes images from any webpage and downloads them into a neat folder.  
Perfect for quick experiments, learning web scraping, or grabbing assets for offline use.

---

##  Features
-  Fetches HTML from a user-provided URL
-  Extracts all `<img>` tags automatically
-  Downloads images into a `downloads/` folder
-  Shows progress with a friendly progress bar
-  Handles errors gracefully (skips broken links)

---

##  Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/kraken-503/jpgsu-dl.git
cd jpgsu-dl
```

### 2. Usage
```bash
python3 cfg.py
```
